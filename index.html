Shared scratch space via kvdb.io buckets. Anyone with this link can read & post here — treat as public. Each space is a single 16 KB value on kvdb.io, so the board keeps only as many recent items as fit in that budget — older ones roll off automatically.

Create or Join a Space

One space is shared by all devices that open its unique link.

kvdb.io (the storage host) requires a real email on new buckets. This app previously fell back to a fake placeholder address when left blank — spaces created that way have been seen getting their writes silently rejected (403 "bucket policy blocked the request"), so a real address is now required.
